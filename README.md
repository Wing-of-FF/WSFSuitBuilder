# WSFSuitBuilder
External Application to Build Armor Suits in Asheron's Call

## 📥 Download
You can download the latest installer from the [Releases](https://github.com/Wing-of-FF/WSFSuitBuilder/releases/tag/Beta) page.

## 🚀 Installation
1. Download the `.msi` file.
2. Run the installer.
3. Follow the on-screen prompts.

## 🛠️ Requirements
* Windows 10/11
* .NET 8 Runtime (or whichever version you used)

* Click "Find Best Suit" will open a new page with the items the program selected.

  * A list of the found Requested Cantrips will be displayed in the top left

  * The suit will be displayed in a grid at the bottom. Scrolling all the way to end you will see a "Copy Stats" button. You do a Ctrl+V in Virindi Global Inventory and click search, it wil find the item for you.
